# Ultimate Tic-Tac-Toe (Jaunt Example)

A complete [Ultimate Tic-Tac-Toe](https://en.wikipedia.org/wiki/Ultimate_tic-tac-toe)
engine — game rules, a minimax AI, and a CLI player — written entirely as **Jaunt specs**
and generated by the Codex engine.

> **This example is also a case study.** It was built end-to-end by a *first-time agent
> with no knowledge of Jaunt's internals* — only the public README. It produced a correct
> implementation with **10/10 generated tests passing on the first run**. See
> [`CASE-STUDY.md`](./CASE-STUDY.md) for the candid blow-by-blow (and the rough edges it
> surfaced).

## What it demonstrates

| File | Jaunt feature |
|------|---------------|
| `src/uttt/game.py` | **Whole-class `@jaunt.magic`** — `UltimateTTT` is a docstring-rich class spec with six method stubs; Jaunt designs and implements the whole class. |
| `src/uttt/ai.py` | **Function `@jaunt.magic` with a cross-module dependency** — `best_move`/`evaluate` (minimax + alpha-beta) that import and use `UltimateTTT`. |
| `src/uttt/cli_player.py` | **`@jaunt.preserve`** — `CLIPlayer.run()` is hand-written and kept verbatim while the other methods are generated (mixed whole-class mode). |
| `tests/test_uttt.py` | **`@jaunt.test`** — ten behavioural test specs (active-board routing, forced redirects, local/meta wins, draw, all the `ValueError` cases). |

## Run it

Requires the `codex` CLI installed and authenticated (`codex login`).

```bash
cd examples/07_ultimate_ttt

# Generate the implementations from the @jaunt.magic specs.
uv run --project ../.. jaunt build

# Generate the pytest battery from the @jaunt.test specs and run it.
uv run --project ../.. jaunt test
```

The generated code lands in `src/uttt/__generated__/` and `tests/__generated__/` (these are
git-ignored — regenerate them with `jaunt build`/`jaunt test`).

## The game, briefly

A 3×3 grid of nine small tic-tac-toe boards. The cell you play *within* a small board
dictates which board your opponent must play in next (the "send" mechanic); if that board
is already won or full, the opponent may play anywhere. Win a small board to claim its cell
on the 3×3 meta-board; win three meta-cells in a row to win the game.
