# TypeScript slugify

This example is generated by Jaunt's TypeScript target with `gpt-5.6-sol`. The
private `src/index.jaunt.ts` contract is never executed or emitted; consumers use
the ordinary `src/index.ts` facade.

```bash
npm ci
export JAUNT_TS_WORKER=../../packages/jaunt-ts/dist/worker/main.js # local checkout only
uv run --project ../.. jaunt sync --language ts
uv run --project ../.. jaunt build --language ts
uv run --project ../.. jaunt test --language ts
npm run typecheck
npm run demo
```

The demo compiles with the project-local TypeScript compiler and runs emitted
JavaScript. The emitted program has no Jaunt runtime dependency.

Installed users omit `JAUNT_TS_WORKER`; Jaunt resolves the worker from the direct
`@usejaunt/ts` dev dependency. The override is needed here because npm links this
example's `file:` dependency back to the sibling package checkout.
