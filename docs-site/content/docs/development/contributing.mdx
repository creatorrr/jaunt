From the repo root:

```bash
uv sync
uv run ruff check .
uv run ty check
uv run pytest
```

## Pre-commit Hooks

Enable repository hooks once from the root:

```bash
git config core.hooksPath .githooks
```

The hook runs:

- `ruff check .`
- `ruff format --check .`
- `ty check`
